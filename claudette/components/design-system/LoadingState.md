## Purpose
Displays a spinner with a loading message and optional subtitle for async operations.

## Imports
- **Stdlib**: none
- **External**: `react`, `react/compiler-runtime`
- **Internal**: `Box`, `Text`, `Spinner`

## Logic
Renders a spinner alongside the loading message in a row, with an optional subtitle below in a column layout. Supports bold and dimColor styling for the message.

## Exports
- `LoadingState` - shows a spinner with a configurable loading message and optional subtitle
