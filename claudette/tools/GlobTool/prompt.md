# ```prompt```

## Purpose
Defines the name and description for the Glob tool that performs fast file pattern matching.

## Imports
- **Stdlib**: None
- **External**: None
- **Internal**: None

## Logic
Exports the Glob tool's name and a bulleted description covering its capabilities: fast pattern matching for any codebase size, glob pattern support, sorted results by modification time, use cases for finding files by name patterns, and guidance to use the Agent tool for open-ended searches requiring multiple rounds.

## Exports
- `GLOB_TOOL_NAME` - Constant string 'Glob' used as the tool's identifier
- `DESCRIPTION` - Multi-line description covering capabilities, pattern support, result sorting, use cases, and when to use the Agent tool instead
