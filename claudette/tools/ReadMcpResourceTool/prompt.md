# ```prompt```

## Purpose
Provides the description and prompt text for the Read MCP Resource tool that reads resources from MCP servers.

## Imports
- **Stdlib**: None
- **External**: None
- **Internal**: None

## Logic
Defines static description and prompt strings that explain the tool's purpose, required parameters (server name and resource URI), and usage examples for the inference provider.

## Exports
- `DESCRIPTION` - brief description of the tool's function with parameter list and usage example
- `PROMPT` - detailed prompt explaining when to use the tool and its required parameters
