# claude_code_internal_event

## Purpose
Generated TypeScript types for Claude Code internal telemetry events.

## Imports
- **Stdlib**: (none)
- **External**: (none)
- **Internal**: Generated protobuf types (Timestamp, PublicApiAuth)

## Logic
1. `GitHubActionsMetadata` - GitHub Actions environment info (actor_id, repository_id, owner_id)
2. `EnvironmentMetadata` - runtime environment info (platform, node_version, terminal, package_managers, runtimes)
3. Boolean flags: is_running_with_bun, is_ci, is_claubbit, is_github_action, is_claude_code_action, is_claude_ai_auth
4. GitHub-specific fields: github_event_name, github_actions_runner_environment/os, github_action_ref
5. WSL version, architecture, claude_code_remote_session_id
6. Deployment environment, conductor flag, coworker_type, build_time
7. Local agent mode, Linux distro/version/kernel, VCS, platform_raw
8. Generated by protoc-gen-ts_proto v2.6.1

## Exports
- `GitHubActionsMetadata` - GitHub Actions environment interface
- `EnvironmentMetadata` - runtime environment interface
- (Many more internal event types)
