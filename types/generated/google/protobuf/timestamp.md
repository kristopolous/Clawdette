# timestamp

## Purpose
Generated TypeScript type for Google Protobuf Timestamp, representing a point in time.

## Logic
1. Timestamp represents point in time independent of timezone/calendar
2. Encoded as seconds + nanoseconds at nanosecond resolution
3. Epoch: UTC midnight January 1, 1970 (proleptic Gregorian)
4. Range: 0001-01-01T00:00:00Z to 9999-12-31T23:59:59.999999999Z
5. All minutes 60 seconds; leap seconds "smeared" (24-hour linear smear)
6. Convertible to/from RFC 3339 date strings
7. Generated by protoc-gen-ts_proto v2.6.1
8. Examples provided for POSIX time(), gettimeofday(), Win32 GetSystemTimeAsFileTime(), Java currentTimeMillis/now()
